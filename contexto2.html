Help Superlative

There are two main ways to form comparatives in English:

  • Adding -er:

  • Singular nouns: add -er to the end:

    • big - bigger, hot - hotter


  • Plural nouns: add -er to the end

    • cats - cater, dogs - dogger

  • Using more: Adjectives: add "more" before the adjective

  • happy - more happy, fast - more fast
  • Adverbs: add "more" before the adverb

    • quickly - more quickly, loudly - more loudly
